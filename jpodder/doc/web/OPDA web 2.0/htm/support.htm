The jPodder project is hosted on Sourceforge. The sourceforge project offers facilities for getting support. There is a user forum, Feature requests section and Mailing lists You can get support for jPodder in the following ways:

Mailing lists

Subscribe to the user or developers list. (You can also browse the archives).
