Mime Type - defines the type of a document or attachement so that the application can use the appropriate application to handle it. MIME stands for Mulitpurpose Internet Mail Extension and normally uses this pattern: <document type>/<sub type>. In an application the Mime type is not associated with one or more applications the application can use to present it to the user. As the name suggests this was originally created for emails but is not widely used in the web.
