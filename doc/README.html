Installation:

This is a JAVA application, which support various operating systems. To run the application simply make sure that the Java Runtime Environment is installed. It's easy to check when typing:

java -version

When installed, the command above will reply with the java version on your system, The version should

be 1.4 or higher. otherwise install java from:

http://java.sun.com

On Windows (win2k & winXP): 

1) Unzip the file to a directory of choosing. [dir]

2) Double-click on install.exe or install.jar , or type java -jar install.jar at the prompt.

3) Follow the instructions to  install jPodder on your system 

On Linux, MAC OSX or other JRE supporting OS:

1) UnBzip2 the file to a directory of choosing. [dir]

2) Double-click on install.jar, or type java -jar install.jar at the prompt.

3) Follow the instructions to  install jPodder on your system 

OS Issues:

jPodder was initially developed for win32 platforms. Some of the win32 functions have not been ported to other OS'es. The following functions are not supported or not fully tested. 

1) The system tray function will be disabled if jPodder can not find a systray4j compatible library or if usage of the the library causes errors. 

2) In the settings->media panel you need to select "Do not transfer to media player" as the other to options will only work on win32.

3) The installer (izPack) will create an icon and Start-Menu entries on win-32 only. To run jPodder on non-win32, go to the installation directory and type java -jar jPodder.jar
